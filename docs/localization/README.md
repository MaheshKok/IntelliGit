# Localization Alternatives

- Google Sheets is the preferred low-cost draft translation path for this
  repository.
- Local/offline tools such as Argos Translate, LibreTranslate, Marian, or NLLB
  avoid API cost, but have weaker quality and setup for product/UI strings.
- Weblate, Crowdin, and Transifex support human review and glossary workflows,
  but can add cost or process overhead.
- Google Cloud Translation, DeepL, and Azure Translator suit automation, but
  require explicit approval before addition.
- Native or screenshot review remains required before machine-generated
  translations are called production-quality.
